# URL Shortener

This is a simple URL shortener built using Spring Boot and MySQL.

## Features

- Shorten long URLs
- Redirect to original URLs

## Technologies Used
- Spring Boot
- MySQL
- JPA/Hibernate
- Maven
- Lombok

## Disclaimer
This project is for educational purposes only.

It was meant as a learning exercise to brush up on my Spring Boot and MySQL skills.

It was not intended for production use, and should not be used as such.
